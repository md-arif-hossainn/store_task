# Store-Details-API
In this project I create a simple Android app that displays a list of Store Details(which came from API).
# Tools Used
1. Recyclerview.<br/>
2. Fragment.<br/>
3. Data Binding.<br/>
4. Object Binding.<br/>
5. Singleton Pattern.<br/>
6. MVVM Architecture.<br/>
7. Retrofit.<br/>
8. Coroutine.<br/>
9. ViewModel.<br/>
10. Adapter.<br/>
